To-Do List App (HTML, CSS & JavaScript)
📌 Project Overview

The To-Do List App is a simple and interactive web application developed using HTML, CSS, and JavaScript. It helps users organize their daily activities by allowing them to add, complete, and delete tasks efficiently.

This project is ideal for beginners who want to practice DOM manipulation, event handling, and basic front-end web development concepts.

✨ Features
Add new tasks instantly
Mark tasks as completed
Delete tasks when no longer needed
User-friendly interface
Responsive design
Real-time task management
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
📚 Learning Outcomes

Through this project, I learned:

How to create dynamic web applications
DOM manipulation using JavaScript
Event handling and user interactions
Working with forms and input fields
Styling responsive layouts using CSS
🚀 How to Use
Enter a task in the input field.
Click the Add button to create a task.
Click on a task to mark it as completed.
Click the Delete button to remove a task.
🎯 Project Purpose

The main purpose of this project is to strengthen front-end development skills and understand how JavaScript can be used to create interactive web applications.
